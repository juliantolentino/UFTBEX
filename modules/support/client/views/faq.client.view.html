Educational Standards

Does Florida use Common Core and are these standards integrated into this website?

Yes, Florida has been using Common Core since the 2013-2014 school year. Our database is utilizing CPALMS' list of standards which includes Common Core english language arts and math standards.

What is Common Core

Common Core is a set of standards for kindergarten through 12th grade in english language arts and mathematics" More information can be found on the Common Core FAQ web page which is linked below!

Project Structure

Can I have private projects that are just for me?

Yes, at the top of the create project and edit project page there is an option to make your project public or private. This allows you to protect your ideas as well as having in-progress projects invisible while your working on them.

User Settings

Where can I view/edit my projects?

You can view your projects by clicking on the top right dropdown menu and selecting My Profile. Your projects will show up on the right side of the screen. Clicking the projects name lets you view the project, clicking the button lets you edit the project. (This button doesn't do anything, it is just a sample of what you will see)

Where can I edit my profile settings?

You can view your profile settings by clicking on the top right dropdown menu and selecting Edit Profile.
